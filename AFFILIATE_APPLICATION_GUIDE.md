# VPN 联盟链接申请完整指南

按以下步骤申请各 VPN 联盟，获取推广链接后，把链接发给我，我会帮你写入 `sites/vpn-usa/config.json`。

---

## 申请前准备

- **网站地址**：`https://wordok.top/vpn-usa/`
- **网站描述**：`VPN review and comparison site targeting USA users with 88+ expert articles`
- **流量来源**：勾选 SEO / Organic Search
- **预计月流量**：如实填写（新站可填 500 左右）

> ⚠️ 建议网站已上线 2–4 周后再申请，新站容易被拒。

---

## 一、NordVPN（优先申请，审核最快）

### 步骤

1. 打开：https://affiliates.nordvpn.com
2. 点击「Apply now」
3. 填写表单（网站 URL、描述、流量来源、月流量等）
4. 提交，通常 1–3 个工作日审核

### 获取链接

- 审核通过后登录后台
- 找到推广链接，格式类似：`https://go.nordvpn.net/aff_c?offer_id=15&aff_id=12345`
- 复制**完整链接**发给我

### 需要你提供

```
NordVPN 完整推广链接（含 aff_id）
```

---

## 二、ExpressVPN

### 步骤

1. 打开：https://www.expressvpn.com/affiliates
2. 点击「Join Now」
3. 填写网站信息（同上）
4. 等待审核通过

### 获取链接

- 登录联盟后台
- 找到你的推广链接（可能包含 `a_fid`、`referrer_id` 等参数）
- 复制**完整链接**发给我

### 需要你提供

```
ExpressVPN 完整推广链接
```

---

## 三、Surfshark

### 步骤

1. 打开：https://surfshark.com/affiliates
2. 填写申请（审核相对宽松）
3. 等待审核通过

### 获取链接

- 登录后台获取推广链接
- 可能是 `https://surfshark.com/deal/你的代码` 或带参数的完整 URL
- 复制**完整链接**发给我

### 需要你提供

```
Surfshark 完整推广链接
```

---

## 四、CyberGhost

### 步骤

1. 打开：https://affiliates.kape.com/users/signup/  
   （CyberGhost 属于 Kape 旗下，通过 HasOffers 申请）
2. 或访问：https://www.cyberghostvpn.com/affiliate-program
3. 注册并填写网站信息
4. 等待审核通过

### 获取链接

- 登录 HasOffers / CyberGhost 联盟后台
- 找到 CyberGhost 的推广链接
- 复制**完整链接**发给我

### 需要你提供

```
CyberGhost 完整推广链接
```

---

## 返回链接时的格式

申请完成后，按下面格式把链接发给我即可：

```
NordVPN: https://go.nordvpn.net/aff_c?offer_id=15&aff_id=你的ID
ExpressVPN: https://www.expressvpn.com/order?offer=xxx&a_fid=你的ID
Surfshark: https://surfshark.com/deal/你的代码
CyberGhost: https://www.cyberghostvpn.com/你的推广链接
```

- 可以只提供已通过的，不必等四家都通过
- 我会把链接写入 `sites/vpn-usa/config.json`，全站 VPN 链接会自动更新
