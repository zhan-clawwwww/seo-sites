6. 6. 适当添加相关的图片和视频引用（如适用）
7. 7. 保存文章到对应站点的 posts 目录
8. 8. 运行编译检查: npm run build
9. 9. 提交更改: git add -A && git commit -m 'Daily update'
10. 10. 等待飞书确认后推送: git push origin main
